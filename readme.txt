A WooCommerce extension to accept credit cards payments via Stripe.

== Description ==

For instructions, news and more informations please visit the official [project page](http://mgiulio.info/projects/mg-wc-stripe/).

== Installation ==

1. Install the plugin with the standard WP procedures
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin in the Admin Menu > WooCommerce > Settings > Checkout > Checkout options: mg Stripe page

When the plugin is deleted it will remove all its data from the database.
